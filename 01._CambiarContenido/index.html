Titular de la página

Subtítulo de la página

Un párrafo de texto como otro cualquiera.

¿Cuáles son las ventajas de usar las hojas de estilo?

  • Más fácil de usar: permiten guardar y almacenar los estilos por separado.
  • Más accesibles: permiten modificar el formato de una página web según sean las necesidades del usuario.
  • Más optimizados: La hoja de estilo se almacena en cache y no tiene que descargar la página web una y otra vez.
  • Multidispositivo: permiten configurar la página para diferentes dispositivos, como son móviles, tablets, etc.

¿Cuáles son las distintas formas en que se pueden utilizar las hojas de estilo?

  • Incluir CSS en etiquetas (Estilos CSS inline)
  • Incluir CSS en una página (Hojas de estilo CSS embebidas)
  • Incluir CSS en un archivo aparte (Hojas de estilo CSS externas)

Describe las partes y la sintaxis de una hoja de estilos:

 Selector {  propiedad :  valor ;  }
  • Selector: indica el elemento o elementos HTML a los que se aplica la regla CSS.
  • Declaración: especifica los estilos que se aplican a los elementos. Esta compuesta por una o mas propiedades CSS con sus valores.
  • Propiedad: permite modificar el aspecto de una caracteristica del elemento.
  • Valor: indica el nuevo valor de la caracteristica modificada en el elemento.
